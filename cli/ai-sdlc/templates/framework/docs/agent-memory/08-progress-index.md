# Progress Index

| Requirement | Title | Status | Priority | Last Agent | Last Updated | Links |
| ----------- | ----- | ------ | -------- | ---------- | ------------ | ----- |

## Status meanings

- Draft -> Planned -> Processed -> Implemented -> Evaluated -> Done
- Blocked (with reason; see evaluation/fix-loop artifacts)

## Rules

- Every requirement must have a row.
- Update Last Agent and Last Updated on every phase completion.
- Links should point to key artifacts: requirement folder, plan, evaluation, ADRs.
