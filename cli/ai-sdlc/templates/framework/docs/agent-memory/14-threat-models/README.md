# 14 — Threat models

One overlay per kind. Each captures STRIDE/OWASP coverage, trust boundaries,
data flows, and mitigations. Pick the kind that matches your project (set in
`docs/agent-memory/index.json` `meta.kind`) and customize.

Kinds: web, api, ai, mobile, desktop, cli, library, monorepo, data, automation, infra, docs, generic.
