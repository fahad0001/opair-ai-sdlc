# Supply chain

## SBOM

- Generated on every release (CycloneDX JSON).
- Stored as a release asset and as a build attestation.
- Verified by `agent-mem sbom-check` against the licenses allowlist.

## Provenance

- SLSA build provenance generated by the release workflow.
- Verifiable via `agent-mem provenance-verify <artifact> <attestation>`.

## Dependency hygiene

- Pin all third-party GitHub Actions by commit SHA.
- Renovate/Dependabot opens PRs; review changelog before merge.
- Block postinstall scripts in CI; allow only on a vetted allowlist.
- Lockfile audited via `npm audit --omit=dev` on every PR.

## Maintainer security

- 2FA required on all maintainer accounts.
- Signed commits required on protected branches.
- Publish tokens scoped to a single package; rotated quarterly.
