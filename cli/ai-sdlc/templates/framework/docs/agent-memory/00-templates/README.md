# Templates

Source-of-truth templates that the **Init** agent (and `agent-mem new`) copy into new
requirement / plan / evaluation folders.

## Layout

| Path                    | Used by                                     |
| ----------------------- | ------------------------------------------- |
| `requirement-skeleton/` | Copied to `02-requirements/R-XXXX/` on Init |

> Plan templates live at `docs/agent-memory/03-plans/_templates/`.
> Evaluation templates live at `docs/agent-memory/05-evaluation/_templates/`.
> ADR template lives at `docs/agent-memory/06-decisions/ADR-template.md`.
